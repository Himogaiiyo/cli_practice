# cli_practice

ブラウザで動くシンプルなカウンターアプリです。

## 使い方

1. このリポジトリを開きます。
2. `index.html` をブラウザで開きます。
   - 例: ファイルをダブルクリック
   - 例: ローカルサーバーを使う場合は `python3 -m http.server 8000` を実行し、`http://localhost:8000/index.html` にアクセス
3. 画面のボタンで操作します。
   - `+1`: カウントを1増やす
   - `-1`: カウントを1減らす
   - `リセット`: カウントを0に戻す

## 仕様

- 外部ライブラリなし、`index.html` 単体で動作
- カウント値を大きく表示
- カウント値は `localStorage` に保存されるため、ブラウザを閉じても保持されます
